FAQ

Questions Answered.

What is Agent Architect?
An interactive questionnaire that interviews you about your AI agent — mission, personality, tools, memory, operations — then generates a structured mega-prompt. Paste that into Claude, ChatGPT, or any AI, and it builds 7 production-grade workspace files for your agent. Think of it as a brain transplant for your AI.
What's the difference between Free and the $17 version?
Free gives you the questionnaire and a mega-prompt you copy into any AI. The $17 version uses Claude directly to generate your files — better output, branching logic based on your agent type, and a complete example workspace from a production agent (Acrid himself) that you can study and steal from.
Is this better than just prompting ChatGPT myself?
Yes. The difference is structure. Most people tell ChatGPT "be my assistant" and get generic output. Agent Architect asks 40+ specific questions that force you to define boundaries, failure modes, memory architecture, and quality standards. The output isn't a personality — it's an operating system.
Do I need to be technical?
No. If you can answer questions about what you want your agent to do, you can use this. The hard technical work — structuring the workspace files, writing the system prompts, designing the memory architecture — that's what Agent Architect handles for you.
What AI does this work with?
Any LLM that accepts text input. Claude (recommended — it built the spec), ChatGPT, Gemini, Llama, Mistral. The output is markdown files that work everywhere. If you use Claude Code, the files become your CLAUDE.md and the agent boots from them automatically.
